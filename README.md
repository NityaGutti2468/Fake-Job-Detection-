# Fake Job Detection using NLP

This project detects fake job postings using Natural Language Processing (NLP).  
It preprocesses job descriptions, extracts features, and trains machine learning models to classify postings as real or fake.

## Features
- Text preprocessing (tokenization, stopword removal, lemmatization)
- Feature extraction (TF-IDF, word embeddings)
- Classification models (Logistic Regression, Naive Bayes, Random Forest, LSTM)
- Evaluation metrics (Accuracy, Precision, Recall, F1-score)
